## 第 1 步：在 src/server/api/routers/ 下创建路由文件

#### 例如新建 src/server/api/routers/strategy.ts：

```ts
import { z } from "zod";
import { createTRPCRouter, protectedProcedure } from "@/src/server/api/trpc";

export const strategyRouter = createTRPCRouter({
  // 查询
  getAll: protectedProcedure.query(async ({ ctx }) => {
    return ctx.db.strategy.findMany({
      where: { creatorId: ctx.session.user.id },
    });
  }),

  // 创建
  create: protectedProcedure
    .input(z.object({ name: z.string().min(1) }))
    .mutation(async ({ ctx, input }) => {
      return ctx.db.strategy.create({
        data: { name: input.name, creatorId: ctx.session.user.id },
      });
    }),
});
```

## 第 2 步：在 src/server/api/root.ts 中注册

```ts
import { strategyRouter } from "@/src/server/api/routers/strategy"; // 新增

export const appRouter = createTRPCRouter({
  auth: authRouter,
  project: projectRouter,
  strategy: strategyRouter, // 新增这一行
});
```

## 第 3 步：在前端组件中调用

```ts
// 查询
const { data } = api.strategy.getAll.useQuery();

// 创建
const createStrategy = api.strategy.create.useMutation();
createStrategy.mutate({ name: "新策略" });
```

## 总结

| 步骤 |	文件 |	操作|
|------|-----------|------|
| ① | 写业务逻辑 | src/server/api/routers/xxx.ts | 新建文件，定义 procedures|
| ② | 注册路由 | src/server/api/root.ts | import 并加入 appRouter|
| ③ | 前端调用 | 任意组件 | api.xxx.yyy.useQuery() 或 .useMutation() |

如果只实现第 ① 步忘了第 ② 步，路由就不会生效。这是 T3 唯一需要注意的地方。
