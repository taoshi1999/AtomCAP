文件上传功能可以在本地开发环境中进行测试，但如果你想在部署预览环境中测试文件上传功能，可以跳过配置BLOB_READ_WRITE_TOKEN环境变量和安装@vercel/blob包的步骤，因为部署预览环境已经预先配置好了这些内容。你可以直接创建API路由和前端组件来测试文件上传功能。我个人比较推荐直接使用部署预览环境进行测试，因为这样可以更接近实际生产环境的配置，避免在本地环境中可能遇到的配置问题。无论你选择哪种方式，以下是文件上传功能的开发流程：

### 1. 配置BLOB_READ_WRITE_TOKEN环境变量（如果使用部署预览进行测试，可跳过这一步）
在开发文件上传功能之前，需要先配置BLOB_READ_WRITE_TOKEN环境变量。这个变量用于授权应用程序访问Blob存储服务，以便进行文件的读写操作。在根目录的.env文件中添加以下行：

```BLOB_READ_WRITE_TOKEN=your_token_here
```
请将`your_token_here`替换为实际的令牌值。切记不要将敏感信息直接写入代码库，确保.env文件被正确配置并且不被提交到版本控制系统中。

### 2. 在项目中安装官方包 npm install @vercel/blob（如果使用部署预览进行测试，可跳过这一步）
接下来，在项目中安装官方的Blob存储包。打开终端，导航到项目根目录，并运行以下命令：

```bash
npm install @vercel/blob
```
这个命令会将@vercel/blob包添加到项目的依赖中，使你能够在代码中使用Blob存储相关的功能。

### 3. 创建一个API路由来处理文件上传 (app/api/upload/route.ts)
在 Next.js 中，你需要提供一个服务端的接口来安全地处理上传请求。创建一个新的API路由文件 `app/api/upload/route.ts`，并添加以下代码：
```typescript
import { put } from '@vercel/blob';
import { NextResponse } from 'next/server';

export async function POST(request: Request): Promise<NextResponse> {
  const { searchParams } = new URL(request.url);
  // 从 URL 参数中获取文件名（例如：MiniMax_BP.pdf）
  const filename = searchParams.get('filename') || 'unnamed_file';

  // 确保有文件流
  if (!request.body) {
    return NextResponse.json({ error: 'No file body found' }, { status: 400 });
  }

  try {
    // 调用 Vercel Blob 的 put 方法直接上传
    // access: 'public' 表示上传后生成一个任何人都可以下载的公开链接
    const blob = await put(filename, request.body, {
      access: 'public',
    });

    // 返回成功信息及文件的永久 URL
    return NextResponse.json(blob);
  } catch (error) {
    return NextResponse.json({ error: 'Upload failed' }, { status: 500 });
  }
}
```
这段代码定义了一个POST请求处理函数，它从请求中获取文件名和文件内容，并使用@vercel/blob包的put方法将文件上传到Blob存储。上传成功后，返回包含文件URL的JSON响应。

### 4. 在前端页面中创建一个文件上传组件 (app/components/FileUpload.tsx)

在前端页面中，你需要创建一个文件上传组件来允许用户选择文件并将其上传到服务器。创建一个新的组件文件 `app/components/FileUpload.tsx`，并添加以下代码：

```tsx
'use client';

import { useState, useRef } from 'react';

export default function FileUpload() {
  const inputFileRef = useRef<HTMLInputElement>(null);
  const [blobUrl, setBlobUrl] = useState<string | null>(null);
  const [isUploading, setIsUploading] = useState(false);

  const handleUpload = async (e: React.FormEvent) => {
    e.preventDefault();
    if (!inputFileRef.current?.files?.length) {
      alert('请先选择一个文件！');
      return;
    }

    setIsUploading(true);
    const file = inputFileRef.current.files[0];

    try {
      // 向我们刚才写的后端 API 发送请求
      const response = await fetch(`/api/upload?filename=${file.name}`, {
        method: 'POST',
        body: file,
      });

      const newBlob = await response.json();
      
      // 保存上传成功后的 URL
      setBlobUrl(newBlob.url);
      
      // TODO: 在这里你可以把 newBlob.url 和 projectId 一起保存到你的关系型数据库中
      
    } catch (error) {
      console.error('上传出错:', error);
    } finally {
      setIsUploading(false);
    }
  };

  return (
    <div className="p-6 border border-gray-200 rounded-lg bg-white w-full max-w-md">
      <h3 className="text-lg font-semibold mb-4 text-gray-800">上传尽调论据</h3>
      
      <form onSubmit={handleUpload} className="flex flex-col gap-4">
        {/* 文件选择框 */}
        <input 
          name="file" 
          ref={inputFileRef} 
          type="file" 
          accept=".pdf,.xlsx,.docx"
          className="text-sm text-gray-500 file:mr-4 file:py-2 file:px-4 file:rounded-md file:border-0 file:text-sm file:font-semibold file:bg-blue-50 file:text-blue-700 hover:file:bg-blue-100"
        />
        
        {/* 提交按钮 */}
        <button 
          type="submit" 
          disabled={isUploading}
          className="bg-blue-600 hover:bg-blue-700 text-white font-medium py-2 px-4 rounded-md disabled:bg-gray-400 transition-colors"
        >
          {isUploading ? '正在上传到 Vercel 云端...' : '确认上传'}
        </button>
      </form>

      {/* 成功后的反馈 */}
      {blobUrl && (
        <div className="mt-4 p-3 bg-green-50 text-green-700 rounded-md text-sm break-all">
          ✅ 上传成功！<br/>
          <a href={blobUrl} target="_blank" rel="noreferrer" className="underline font-bold">
            点击这里查看文件
          </a>
        </div>
      )}
    </div>
  );
}
```
这个组件允许用户选择一个文件并将其上传到服务器。上传过程中会显示一个加载状态，上传成功后会显示一个链接，用户可以点击链接查看上传的文件。你还可以在上传成功后将文件的URL和相关的项目ID保存到你的关系型数据库中，以便后续管理和使用。
